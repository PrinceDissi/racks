Basics

Welcome to Racks

Racks is a framework which implements the WebComponents and Shadow DOM standards (drafts) written by the W3C organization. It's supported in the major browsers, which are on the highest stand of implementation of these standards.

This little piece of software is inspired by the Polymer Project and the X-Tags Project which both implement one or more interface(s) to create custom tags.

About Rack

There are a few things which most of the user/developer wants to know before using a framework.

For who or what is Rack?
